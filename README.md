## Supported devices:
### KeyWe GKW-2000D EU Frequency    


## Flow cards support following triggers
1. Lock/unlock
2. Lock/unlock by BLE/Z-Wave
3. Lock/Unlock by button
4. Lock/Unlock by key/thumbturn
5. Unlock by user slot (by pin/nfc)
6. Lock by keypad
7. Auto locked
8. Deadbolt jammed
